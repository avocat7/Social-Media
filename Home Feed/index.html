Hot take: most performance problems in modern web apps aren't caused by JavaScript bundle size — they're caused by layout thrashing and unoptimized render cycles. I've seen 40% perf gains just from fixing paint invalidation. Stop blaming the framework, start profiling properly 🔥

// Before — triggers layout thrash
elements.forEach(el => el.style.height = el.offsetHeight + 'px');
// After — batch reads then writes
const heights = elements.map(el => el.offsetHeight);
elements.forEach((el, i) => el.style.height = heights[i] + 'px');
